# Download the data
Download the the different .tsv files from [ImdB non-commerical datasets](https://developer.imdb.com/non-commercial-datasets/). Unzip them a put them into a folder called **data** at the root of the project. So your project should have a folder structure that looks like this ./data/*.tsv. 

# CreateDataset.ipynb
This file creates the dataset that is used for the machine learning. 

Make sure that the file paths are set correctly to the datasets files.
