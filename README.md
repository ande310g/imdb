# imdb

# Download the data
Download the the different .tsv files from [ImdB non-commerical datasets](https://developer.imdb.com/non-commercial-datasets/). Unzip them a put them into a folder called **data** at the root of the repository. So your repository should have a folder structure that looks like this ./data/*.tsv. 

# Create dataset
This file creates the dataset that is used for the machine learning. 

In the file CreateDataset.ipynb make sure that the file paths are set correctly to the datasets files.
